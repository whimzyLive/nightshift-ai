**Epic:** NA-50 — Public docs generation via /sdlc:docs command + knowledge-engineer

**Agents required:** `ai-enablement-engineer` (sole agent)

**Goal:** Ship the plugin's Diataxis-based doc-type registry (`refs/doc-types.md`), the docs-manifest template (`refs/docs-manifest-template.md`), and an opt-in `/sdlc:init` step that scaffolds `.claude/project/docs-manifest.md` — with zero generation logic and zero runtime manifest consumption.

**Architecture:** Three plugin-source artifacts only — a new data-registry ref, a new template ref, and modifications to the `/sdlc:init` command flow (a Step 3 opt-in prompt, a Step 4g write sub-step, Step 0 merge-jump-list wiring, and a Step 5 checklist line). No product surface. The registry is the single source of truth; the manifest carries only activation state.

**Tech Stack:** Markdown plugin refs + `/sdlc:init` command flow (`plugins/sdlc/**`). No code, no tests-as-code — verification is registry self-check conformance + frontmatter validity.

## Global Constraints (copied from spec — every task inherits these)

- All writes land under `plugins/sdlc/**` only. Do **not** touch `.claude/project/project-context.md`, `refs/project-context-template.md`, other agents' overrides, or any product workspace.
- The consumer artifact `.claude/project/docs-manifest.md` is written by `/sdlc:init` at consumer-repo runtime — it is **never** committed into the plugin.
- Registry rows carry **exactly 8 columns in order:** `type`, `quadrant`, `trigger`, `source-of-truth`, `generation-mode`, `target-path`, `public`, `applies-when`.
- `trigger` tokens ∈ `{sync, release, seed, audit, distill}`, comma-separated **with no spaces** (e.g. `seed,sync`).
- Manifest rows carry **exactly 3 columns:** `type`, `enabled` (default `true`), `target-path`. `trigger`/`generation-mode`/`public`/`quadrant` are **derived at read time from `refs/doc-types.md`** and MUST NOT be copied into the manifest.
- No `<...>` placeholder tokens may remain in any written consumer file (existing `/init` Step 4 rule).
- Do **not** abstract a shared registry format — copy the shape of `refs/skills-map.yml`; do not generalize.
