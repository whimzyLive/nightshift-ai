# Site-wide night-sky chrome and design tokens — Implementation Plan

**Spec:** [docs/superpowers/specs/NA-30.md](../specs/NA-30.md)
**Story:** NA-30 — Add site-wide night-sky chrome and design tokens
**Epic:** NA-29 — Nightshift marketing site — design-handoff build
**Date:** 2026-07-12
**Agents required:** web-engineer (only)

## Execution order

Single phase — **Phase 4 (web-engineer)** only. This story is web-only: the spec declares
**no Data Model, no API Surface, no offline-sync, and no Mobile UI**, so Phases 1–3 and 5
(database-administrator, platform-engineer, sync-engineer, mobile-engineer) have no tasks. No
`plugins/`, `skills/`, or AI-config surface is touched, so ai-enablement-engineer has no tasks
either. All work runs on the shared `feat/NA-30` branch.

Tasks below are ordered by dependency: Tailwind/PostCSS pipeline → `@theme` tokens + base layer →
fonts + logomark assets → leaf primitives (no chrome deps) → chrome components → layout shell +
routes → visual-parity QA. Each task names exact files/tokens/routes from the spec so it is
completable without questions.

## Global constraints (apply to every task)

- **TSX only** — all UI is `.tsx`, no `.jsx`/`.js`/class components (root `CLAUDE.md`).
- **Tailwind v4 CSS-first** — no `tailwind.config.js`; config lives in CSS via `@import "tailwindcss"`, `@theme`, `@source`.
- **No CSS Modules / SCSS** anywhere new — do **not** touch `apps/marketing/src/app/(payload)/custom.scss`.
- **Sharp-edges rule** — every rectangle (cards, buttons, pills, badges, nav, inputs) renders `border-radius: 0`, intentionally overriding `--radius-*`. Only genuinely circular elements (star dots, gate circles, traffic-light dots, avatar discs) stay round.
- **Tokens are authoritative** — copy values verbatim from `docs/design/marketing-site-handoff/tokens/*.css`; never re-derive or round. `--text-on-accent` = `#f5f3ef` (site override).
- **Reduced motion** — `@media (prefers-reduced-motion: reduce)` statically disables all motion; content stays legible and reachable; focus-visible rings (`--glow-focus`) remain in all cases.
- **Component home** — brand primitives in `packages/ui/src/lib/nightshift/`, re-exported from `packages/ui/src/index.ts`; app composition (layout, routes, fonts, `global.css`/`@theme`) in `apps/marketing`.
- **Scope guard** — chrome + tokens only. Do **not** build `terminal`, `pipeline`, `install-snippet`, `agent-card`, any page body content, CMS, SEO/JSON-LD, or the home-only heavy sky (moon/meteors/parallax).
