# Changelog

Nightshift releases are generated automatically from [Conventional Commit](https://www.conventionalcommits.org/) messages by `nx release`. To avoid a second, drifting source of truth, this page does not duplicate them — it points to where the generated notes live.

## Canonical release notes

- **GitHub Releases** — published per tag:
  - [sdlc@0.45.0](https://github.com/whimzyLive/nightshift-ai/releases/tag/sdlc%400.45.0)
  - [gtm@0.5.1](https://github.com/whimzyLive/nightshift-ai/releases/tag/gtm%400.5.1)
- **In-repo changelogs** — the full cumulative history per plugin, regenerated on every release:
  - [`plugins/sdlc/CHANGELOG.md`](../../plugins/sdlc/CHANGELOG.md)
  - [`plugins/gtm/CHANGELOG.md`](../../plugins/gtm/CHANGELOG.md)

A dedicated docs-site release page that mirrors these automatically on each release is planned; until then, the sources above are authoritative.
