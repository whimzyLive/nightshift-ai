### N1 · Wordmark + 2 links

Top-of-page bar: wordmark on the left, two text links on the right ("Pricing" / "Sign in"). No logo image, no menu icon.
_Use when:_ the page has very few destinations.
_Don't confuse with:_ N3 Side-rail (which is vertical).

```html
<nav class="nav-min">
  <a class="wordmark">Studio</a>
  <ul>
    <li><a>Pricing</a></li>
    <li><a>Sign in</a></li>
  </ul>
</nav>
```
